Modifié la variable d'environnement PATH
 - ds explorateur de fichier : click droit sur Ordinateur
   -> Propriétés -> Paramètres système avancés -> Variables d'Environnement
 - trouver PATh dans la liste déroulante et Modifier
 - ajouter le chemin jusqu'au compilateur mingw dans cette variable
   ex; Avant ds PATH C:\Windows\system32;C:\Windows;C:\Windows\System32\Wbem;
       Après ds PATH C:\Windows\system32;C:\Windows;C:\Windows\System32\Wbem;F:\Programmes\CodeBlocks\MinGW\bin;
	   
Installer les source de wxWidget 
 - sur https://www.wxwidgets.org/downloads/
 - prendre Latest Stable Release 3.0.2, Windows Installer
 - installer où tu veux, on va dire D:\Chemin\vers\wxWidget
 
Ouvrir une console de commande (en mode administrateur si wxWidget est installé à un endroit
   où seul l'administrateur peut écrire)

Aller dans D:\Chemin\vers\wxWidget\build\msw

Ouvrir le fichier config.cpp pour modifier 1 truc, et sauvegarder
# Standard flags for C++ 
CXXFLAGS ?= -std=gnu++11
# Build quality assurance classes library (USE_GUI must be 1)? [0,1]
USE_QA ?= 1

Dans la console, compiler en faisant
mingw32-make -f makefile.gcc

Attendre :o)

Si ça compile, c'est déjà un super bon point. Il ne reste plus qu'à dire à codeblock d'utiliser cette libraire :o)
